# College AI Assistant

An AI-powered assistant designed to help students with college-related queries, perform quick calculations, and get real-time date and time updates. Built using Python and Transformers, this project showcases practical AI integration in everyday tools.

## Features
- Get information about CBIT courses, clubs, NSS, and placements.
- Perform instant calculations using natural input.
- Check current date and time.
- AI text generation for general queries (powered by DistilGPT-2).

## Tech Stack
- Python
- Transformers (Hugging Face)
- Regex & Datetime modules
